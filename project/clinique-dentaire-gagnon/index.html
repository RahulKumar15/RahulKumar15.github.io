Clinique dentaire Gagnon

  • Un design épuré et élégant pour la Clinique dentaire Gagnon

    La Clinique dentaire Gagnon offre des services de dentisterie pour toute la famille. Le design du site Web devait refléter l’élégance de l’image de marque de la Clinique. Conçu avec le CMS WordPress, les contenus du site peuvent facilement être modifiés par le client afin de garder l’offre de services à jour. Dans le développement de ce site, nous avons concentré nos énergies dans l’intégration de transition de pages esthétiques afin que l’expérience client soit améliorée.

    Mentions récoltées par le projet :

    • Awwwards : Honorable mentions ;
    • CSS Design Awards : Special Kudos, Innovation, UI Design, UX Design. 

    Les principales caractéristiques du projet :

    • CMS : WordPress ;
    • Responsive ;
    • Expérience client améliorée : Motion UX.
